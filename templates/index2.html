How To Use

Using this Web Application is simple and easy.

  • Get Started

    Navigate to the Prediction Section

    In the navigation bar of the web page click on predition, and get redirected to the prediction section.

  • Next Step

    Upload File

    After getting redirected, Select an image from local or paste a web link to an image, then click submit to send it to our server for processing.

  • Final Goal

    Hold onto your nerves

    After uploading image or submitting any web images via link wait a while till model returns the predicted caption on the web page.

Some Words About Our Project

How It Works

The MultiModel neural network architecture that brings the CNN and LSTM models into one has achieved state-of-the-art results on image caption.

Model Architecture

Language Model and Feature Extraction

We stop the caption generation process when the language model emits an END token or the length of the caption increases a threshold. In the demo, this threshold has been set to 40 which is the maximum length of captions in Flickr8k dataset.

  • The Feature Extractor : which takes an image as input and outputs a low dimensional condensed repersentation of the image.
  • The Language Model : which takes the condensed representation and a special START token and generates a caption

Model Summary

Final Model

Trained Image model and Caption model will be merged via Keras to perform predictions.
